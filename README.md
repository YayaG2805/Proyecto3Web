# Mi Bitacora de Entrenamiento

Proyecto final de la Fase 1 para el curso Sistemas y Tecnologias Web.

## Descripcion inicial

Mi Bitacora de Entrenamiento sera una aplicacion para registrar sesiones de entrenamiento fisico. En esta fase se trabajaran dos piezas independientes:

- Frontend React con Vite, funcionando con LocalStorage.
- Backend Express con PostgreSQL, funcionando como API REST separada.

La conexion entre frontend y backend se realizara en la Fase 2.

## Tema elegido

Entrenamiento fisico.

## Arquitectura base

- Item: Sesion de entrenamiento.
- Registro: Minutos de sesion.
- Categorias: Fuerza, Cardio, Flexibilidad, Deportes.

## Estado del proyecto

Estructura base creada para iniciar la Fase 1 por bloques.
